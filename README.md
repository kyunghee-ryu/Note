<h2>UWBRFTestSolution 사용방법 안내입니다. </h2>

- 실행파일을 더블클릭해서 실행하셔도 되고,  
  command 를 확인하고 싶으신 경우 실행파일을 콘솔창에서 -console 파라미터를 주어 실행하십시오. 
 
- 왼쪽 드랍다운 박스를 클릭하셔서 COM 포트를 지정하시고 connect 하십시오. 

- Anchor setting 의 왼쪽 Rx Port 와 Channel 을 선택하시고, 오른쪽의 측정 항목을 선택하신 후 Init 하여 특정합니다. 
  Tx 와 Rx 모두 Stop 버튼을 누를 때 까지 반복합니다. 
	
- 좌측 아래 Parameter Setting 의 설정값들은 프로그램 종료시 저장되고, 다음 실행시 불러오도록 하였습니다.
